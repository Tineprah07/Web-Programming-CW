About This App

Timer App is a web application designed to accurately time races and record finish positions for up to 500 participants. It was built by student up2158902 as part of the Web Programming coursework (M30237) at the University of Portsmouth.

The app offers an intuitive interface optimized for outdoor use, even in cold or low-light conditions. Key features include:

  • Start and pause a race timer with millisecond precision
  • Record and store finish times locally, even while offline
  • Submit and export race results to a server or as CSV
  • Night Mode for improved readability in dark environments
  • Designed for usability with gloves and minimal interaction
