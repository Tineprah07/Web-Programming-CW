About This App

Timer App is a race timing tool designed for events like the Pub-to-Pub race hosted by the Portsmouth Joggers' Club. Built as part of the Web Programming coursework (M30237), the app enables:

  • Starting and stopping a race timer
  • Recording finish times for participants
  • Submitting results (even offline)
  • Exporting and deleting race data
  • Night Mode for improved visibility
